# Healthy Habit Planner

## Purpose
A client-side web application that allows users to create, track, and complete healthy habits using accessible forms and persistent localStorage data.

## Features Implemented
- Create new habits
- View habit list
- Mark habits complete
- Data persistence using localStorage
- Accessible form controls and semantic HTML
- Simple habit streak tracking

## Accessibility
- Semantic HTML elements used
- All form inputs labeled
- Keyboard navigation supported
- Visible focus states

## Data Storage
All data is stored locally in the browser using localStorage. No external services are required.

## Future Improvements
Due to time constraints, the following planned features were not implemented:
- Calendar view
- Charts and analytics
- Import/export functionality
- User settings and themes
